# 🎁 Jogo do Amigo Secreto

Este projeto é uma aplicação simples em **JavaScript**, **HTML** e **CSS** que permite cadastrar amigos em uma lista e realizar o sorteio de um **amigo secreto**.

## 📌 Funcionalidades
- Adicionar amigos digitando seus nomes.
- Exibir os amigos cadastrados em uma lista.
- Sortear aleatoriamente um amigo da lista.
- **Validação de entrada:**
  - Impede adicionar nomes vazios.
  - Impede adicionar entradas numéricas (ex: "123").
  - Exibe um alerta com a mensagem **"Por favor, insira um nome."** nesses casos.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: estrutura da aplicação.
- **CSS3**: estilização e layout responsivo.
- **JavaScript (ES6+)**: lógica do jogo.

## 📂 Estrutura do Projeto
```
.
├── index.html      # Estrutura principal da página
├── style.css       # Estilos da aplicação
├── app.js          # Lógica em JavaScript
├── assets/         # Imagens utilizadas
```

## 🚀 Como Executar o Projeto
1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo **index.html** em qualquer navegador.
3. Digite os nomes dos amigos e clique em **Adicionar**.
4. Quando a lista estiver pronta, clique em **Sortear amigo** para ver o resultado.

## 📸 Demonstração
- Campo para digitar nomes.
- Lista dinâmica com os amigos cadastrados.
- Resultado do sorteio exibido abaixo da lista.

## 📖 Exemplo de Uso
- Digite: `Ana`, clique em **Adicionar**.
- Digite: `Carlos`, clique em **Adicionar**.
- Digite: `Marina`, clique em **Adicionar**.
- Clique em **Sortear amigo** → O sistema escolherá um nome aleatório.

⚠️ Se o campo estiver **vazio** ou se você digitar apenas **números** (ex: `123`), aparecerá o alerta:  
**"Por favor, insira um nome."**
